#### 水源水质监测仪
| 字段名称 | 单位 | 说明 |
|---------|------|------|
| 水源浊度 | NTU | 原水含沙量/悬浮物 |
| 水源pH值 | - | 原水酸碱度 |
| 水源水温 | ℃ | 原水温度 |
| 溶解氧（DO） | mg/L | 原水含氧量 |
| 氨氮 | mg/L | 原水污染指标 |
| 耗氧量（CODmn） | mg/L | 原水有机物含量 |
| 总磷/总氮 | mg/L | 富营养化指标 |
| 藻类密度 | 个/L | 藻类预警（地表水） |
